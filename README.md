## 🔐 About Caesar Cipher

The Caesar Cipher is a classical encryption technique where each letter in a message is shifted by a fixed number of positions in the alphabet.

Example with shift value **3**:

HELLO → KHOOR

Each letter moves **3 positions forward** in the alphabet.

## ⚙️ Features

* Encrypt text messages
* Decrypt encrypted messages
* User input for message and shift value
* Supports both uppercase and lowercase letters
* Keeps spaces and special characters unchanged
